# netspeed

## Install

### `cargo`

```console
# install
$ cargo install netspeed

# update
$ cargo install --force netspeed
```

### MacOS

```console
$ brew tap ymgyt/netspeed
$ brew install netspeed
```


## Usage

```console
$ netspeed                                                                                                                                                     +[master]
INFO  Connecting to "netspeed.ymgyt.io:5555"
INFO  Start downstream duration: 3 seconds
```
